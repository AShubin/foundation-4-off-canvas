Foundation 4 Off-Canvas Navigation

This is a simple proof of concept application of Tom Rich's Super simple off-canvas navigation, albeit with a few slight modifications.

The Recipe

  • This example is built on a generic Compass install of Foundation [4.3.2]. In fact, this page is a slightly modified version of the auto-generated index.html page.
  • The Font Awesome library is already linked in the head of this page. It is used to generate the navicon.
  • The jQuery in Tom's example has been slightly modified, and can be found just above the closing body tag of this page.

Getting Started

Simply download or fork the code on github, explore the individual pieces, or compare this example with Tom's example.
